Portfolio CMS

A custom static-site generator with a modern dashboard.

Overview

This project is a lightweight portfolio CMS that generates static pages based on an internal JSON structure. It allows easy content updates without editing any HTML manually.

Features

Custom Dashboard

Manage content visually with a clean, minimalist UI.

Static Export

Outputs pure HTML/CSS/JS files for fast hosting anywhere.

Markdown Support

Write content in Markdown and instantly convert it.

Tech Stack

HTML, CSS, JavaScript, JSON-based content schema, optional Node export tools.
